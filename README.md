## AI-Powered Song Lyric Generator & Composer Assistant

SunoForge is a comprehensive song creation tool that uses AI to generate professional-quality lyrics, suggest chord progressions, and provide detailed production guidance. Designed for use with Suno AI or other music creation platforms.

Supports **Google AI Studio** (Gemini models), **OpenRouter** (hundreds of third-party models), and **local OpenAI-compatible LLM servers** as AI providers. No installation required — runs entirely in your browser.

---

## Key Features

- **Generate from Scratch** — Start with a title or concept; AI generates complete lyrics, style prompts, and chord progressions
- **Analyze & Import** — Paste existing lyrics or a style description to auto-configure all settings
- Complete lyrics with professional song structure and Suno-compatible meta tags
- Suno-compatible style prompts (genre, mood, era, production)
- Suggested chord progressions by section
- Export-ready format for immediate use in Suno
- Full creative control: genre, mood, structure, vocals, instruments, tempo, key, duration, and more
- Regenerate or manually edit individual song sections without losing the rest
- Song history with per-song save, load, export, and full backup/restore
- 11 UI languages (English, German, French, Dutch, Spanish, Portuguese, Russian, Japanese, Korean, Simplified Chinese, Traditional Chinese)
